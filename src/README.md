# Expense Tracker

A full-stack expense tracking application with a React frontend and Spring Boot backend.

## Tech Stack

### Frontend
- React 18
- React Router
- Recharts
- React Icons
- Create React App (react-scripts)
- Custom CSS

### Backend
- Spring Boot 3.2
- Spring Security with JWT
- Spring Data JPA
- PostgreSQL
- Maven

## Features

### Backend (REST API)
- User registration and login with JWT authentication
- User profile management (update profile, theme preference, avatar)
- Category management (create, read, update, delete)
- Expense management (create, read, update, delete; filter by date range; group expenses; expense totals)
- Group management (create, read, update, delete; member management)

### Frontend (UI)
- Landing page with hero, features, and how-it-works sections
- Login and signup pages
- Dashboard with spending charts (pie and line), budget overview, savings goals, and recent transactions
- Add and edit expense forms with validation
- Transaction history with search, filtering, sorting, pagination, and edit/delete
- Group expense page with split balance summary, activity feed, and group chat
- Settings UI for profile, theme, and custom categories
